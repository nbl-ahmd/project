# Data Setup

Dataset and generated crop folders are intentionally not committed.

Use local folders such as:

- `data/raw` for full prescription images
- `data/doctors-handwritten-prescription-bd-dataset` for the public BD cropped-word dataset
- `data/processed` for generated page, region, line, and word crops
- `data/custom_word_ocr_dataset` for reviewed word-level OCR exports

The notebooks and scripts in this repository recreate the processed folders from local data.
